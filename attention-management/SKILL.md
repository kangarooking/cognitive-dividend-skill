# 注意力管理

## R — 原文 (Reading)

> "你的一切价值创造活动，最终都是由你的注意力交换得来的！想一想，你唯一与生俱来的、可以自主控制的并且还能拥有生产力的，除了注意力，还有什么？没有了！注意力，就是你拥有的最宝贵财富！"
>
> "将注意力尽量都花在自我成长上，你将会获得最高的投入产出比，这也许是你这一生，听到过的最重要的一条建议！"
>
> —— 《认知红利》第1章

## I — 方法论骨架 (Interpretation)

1. 注意力 > 时间 > 金钱：注意力是唯一与生俱来、可自主控制、具有生产力的资产；时间和金钱只是注意力的衍生品
2. 注意力有三种黑洞：(1) 浪费 — 射向无关事物的能量（八卦、热搜、围观）；(2) 被收割 — 注意力商人用免费内容吸引你再打包卖给广告商；(3) 被利用 — 大数据分析你的注意力轨迹，定向投喂，持续收割
3. 注意力应投向四个高回报方向：(1) 聚焦价值创造（心流工作）；(2) 人际关系尤其是亲密关系；(3) 寻找时代趋势（顺势而为）；(4) 自我成长（最高投入产出比）
4. 不管理注意力的后果：被人替你管理，性格和行为能力受损，人生天差地别

## A1 — 书中应用 (Past Application)

- 作者用"看公众号一眼就帮对方增加0.8元收入"的例子量化了注意力的商业价值
- 用网红与打工人的收入差距说明注意力被集中后的巨大力量
- 引用注意力缺陷障碍（ADD）调查数据（75%违法青少年学龄期患有ADD），证明注意力管理的严重性
- 以特朗普2016年大选利用社交网络和大数据定向投放为例，展示注意力被利用的危险

## A2 — 触发场景 (Future Trigger)

**何时使用此 skill：**
- 每日复盘时发现"今天好像什么也没做，但很累"
- 发现在短视频/社交媒体上连续消耗超过30分钟且无明确目的
- 工作中频繁被打断，注意力碎片化，效率极低
- 感觉自己在"瞎忙"，但产出不成比例

**语言信号：**
- "我时间都去哪了"
- "又刷了一晚上手机"
- "为什么这么忙却没产出"
- "管不住自己看手机"
- "信息焦虑"

**与相邻 skill 区分：**
- 与 `metacognition-control` 的区别：本 skill 聚焦"注意力该投向哪里"的战略层判断；metacognition-control 聚焦"如何控制大脑处理信息"的操作层执行
- 与 `time-merchant` 的区别：本 skill 管理注意力的分配方向；time-merchant 关注如何将时间卖出更高价格
- 如果问题是"我知道该做什么但管不住大脑"→ 用 metacognition-control
- 如果问题是"我怎么才能赚更多钱"→ 用 time-merchant

## E — 可执行步骤 (Execution)

**步骤1：审计注意力去向**
- 连续3天，每小时记录一次注意力的实际投向（可用手机备忘录）
- 第三天晚上将记录分类：浪费/被收割/被利用/有价值
- 完成标准：产出一周注意力去向清单，量化四类各占比

**步骤2：设置注意力防火墙**
- 列出你的注意力黑洞清单（具体App、网站、场景）
- 针对每项设定阻断措施：卸载/限时/物理隔离
- 在四个高回报方向中，为每个方向设定每日最低注意力投入时长
- 完成标准：黑洞清单有明确阻断措施，四个方向各有具体时间承诺

**步骤3：建立注意力分配惯例**
- 每日晨间用5分钟规划今天的注意力分配（哪些任务、哪个方向）
- 每晚用5分钟复盘：今天注意力实际去向 vs 计划去向
- 每周调整一次四个方向的时间比例
- 完成标准：连续7天完成晨间规划+晚间复盘，注意力浪费比例较审计期下降50%以上

## B — 边界 (Boundary)

**不要用的场景：**
- 正在处理紧急危机时不要做注意力审计（先处理危机）
- 创意工作者在灵感爆发期不要强行阻断"看似无目的"的浏览（可能是潜意识在寻找趋势）
- 与家人相处的放松时间不要用生产性思维衡量每分钟的"价值"

**失败模式：**
- 把"管理注意力"变成另一种焦虑源，反而消耗更多注意力
- 过度隔绝信息，导致错过真正的趋势信号（第三方向的"寻找新趋势"需要一定量的信息输入）
- 只关注"价值创造"方向而忽视"人际关系"方向，导致亲密关系恶化

**作者盲点：**
- 对"自我成长"的定义过于宽泛，没有区分有效学习与伪学习（如买课不看、收藏不看）
- 四个方向之间可能存在资源争夺，缺乏优先级排序的决策框架
- 未充分考虑不同职业阶段注意力分配策略的差异（新手期 vs 成熟期）

## 相关 skills

本 skill 与以下 skill 存在关联：
- **metacognition-control**（协同）：注意力管理决定注意力投向哪里（战略层），元认知控制决定大脑如何处理这些注意力捕获的信息（操作层），两者协同构成从"注意力分配"到"信息处理"的完整链路。
- **time-merchant**（协同）：注意力管理确保注意力不被浪费，时间商人框架则将注意力转化为可售卖的时间价值，两者配合可实现注意力→时间→收入的高效转化。
