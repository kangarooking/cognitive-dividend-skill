# 时间商人四模式

## R — 原文 (Reading)

> "个人生产总值=时间单价×能产生价值的时间"
>
> "我们每个人在这个世界上，其实都在经营同一种产品，叫作'时间'，而我们每个人都是'时间商人'。人与人之间收入有如此大的差距，看似是价值之争，其实是模式之争。"
>
> —— 《认知红利》第2章

## I — 方法论骨架 (Interpretation)

1. 核心公式：个人生产总值 = 时间单价 × 能产生价值的时间。收入差距的本质不是努力程度的差距，而是经营模式的差距
2. 四种时间经营模式，从低级到高级：
   - **零售时间**：把时间单份售出（打工）。提升路径是提高时间单价（专业能力）+ 管住注意力（增加有效工作时间）。瓶颈：一天只有24小时
   - **批发时间**：把同一份时间卖给尽可能多的人（直播、写书、录课）。关键：善用互联网技术将边际成本降为零。突破时间总量的物理限制
   - **买卖时间**：购买他人的时间来放大自己的价值（创业雇人、外包）。本质是放大器：买入时间提升效率、提高单价、扩大规模。前提是你自身必须有价值
   - **收时间税**：建立平台让人们在上面交易时间，从中抽税。平台不是原因而是结果，从单点出发积累用户
3. 升级路径不是线性打怪，而是识别当前模式并寻找跃迁机会

## A1 — 书中应用 (Past Application)

- 作者用同学（程序员加班拼命）与网红（20万粉丝直播）的收入对比，量化展示零售 vs 批发的差距：同学个人生产总值约800元/天，网红约20000元/小时
- 用回转寿司的比喻说明时间交易的本质：每个人都像寿司师傅，用知识、能力把流过自己身边的时间餐盘装满食物出售给别人
- 用"唐骏年薪上亿"和"NBA球星"说明零售模式中时间单价的提升空间
- 用公众号作者招团队（编辑、运营、商务）的案例说明买卖时间模式的具体操作

## A2 — 触发场景 (Future Trigger)

**何时使用此 skill：**
- 感觉自己收入遇到瓶颈，加更多班也无法突破
- 想评估自己当前属于哪种时间经营模式，以及升级方向
- 考虑是否要开始做内容、做产品、做平台时需要商业模式判断
- 想理解为什么"同样努力的人"收入差距可达数百倍

**语言信号：**
- "我很努力但收入没增长"
- "为什么别人没那么忙却赚得更多"
- "如何突破收入天花板"
- "我该怎么提高自己的时间价值"
- "做副业有没有意义"
- "要不要创业"

**与相邻 skill 区分：**
- 与 `attention-management` 的区别：本 skill 聚焦"你的时间值多少钱以及如何卖出更高价"；attention-management 聚焦"注意力该投向哪里"
- 与 `compound-effect` 的区别：本 skill 是经营模式的选择框架；compound-effect 是如何在选定模式中实现指数增长
- 如果问题是"我每天很忙但不知道忙了什么"→ 用 attention-management
- 如果问题是"我已经选好了模式，怎么让增长加速"→ 用 compound-effect

## E — 可执行步骤 (Execution)

**步骤1：诊断当前模式**
- 计算你的时间单价：月收入 ÷ (每月工作天数 × 每天有效工作时间)
- 判断你当前属于四种模式中的哪一种（可能混合多种，找出占比最大的）
- 量化你的"能产生价值的时间"占全部时间的比例
- 完成标准：写出现状诊断表（当前模式、时间单价、有效时间占比、月生产总值）

**步骤2：选择升级方向**
- 零售→批发：你拥有的哪些技能/知识可以产品化（写作、录课、直播、模板）？
- 零售/批发→买卖：你能否通过雇佣/外包把低价值时间腾出来？
- 批发/买卖→收税：你是否在一个细分领域有足够影响力来搭建平台？
- 列出至少1个可行的模式升级方案，附时间表
- 完成标准：产出一个模式升级计划，包含目标模式、具体路径、3个月里程碑

**步骤3：执行升级实验**
- 用最小可行产品（MVP）验证升级方向：如写一篇文章测试批发、招一个兼职测试买卖
- 运行30天，记录实际数据：新模式的收入、时间投入、边际成本
- 对比实验前后"个人生产总值"的变化，决定是否全面切换
- 完成标准：30天实验报告，包含数据对比和 go/no-go 决策

## B — 边界 (Boundary)

**不要用的场景：**
- 不要在职业早期（0-3年）就急于跳到"收时间税"模式，基础能力和资源不够，大概率失败
- 不要把"做内容"简单等同于"批发时间"——如果没有真正有价值的内容，只是换了一种形式的"浪费时间"
- 不要为了"升级模式"而裸辞——在零售模式中积累的能力和资源是升级的基础

**失败模式：**
- 把"提高时间单价"理解为多考证书、多读学位，忽略了市场真实需求
- 批发模式中追求数量牺牲质量，导致个人品牌贬值
- 买卖模式中"本身价值很低却大量雇人"，结果放大的是无能而非能力
- 收税模式中把"做平台"当目标而非结果，没有从单点价值积累开始

**作者盲点：**
- 四种模式的升级不是简单的阶梯关系，现实中很多成功案例是跳跃式的
- "收时间税"模式的讨论过于简略，忽略了平台竞争的残酷性和资本壁垒
- 未充分讨论模式切换过程中的过渡期风险（收入断档、能力错配）
- 对"能产生价值的时间"的计量过于理想化，现实中有效时间很难精确统计

## 相关 skills

本 skill 与以下 skill 存在关联：
- **attention-management**（协同）：注意力管理保证时间的有效利用，时间商人框架则将有效时间转化为更高的商业价值，两者协同实现"注意力→有效时间→更高收入"的链路。
- **compound-effect**（协同）：时间商人帮你选择经营模式，复利效应帮你在选定模式中设计指数增长，两者配合可让收入从线性增长跃升为指数增长。
