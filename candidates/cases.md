## case-c01: 鞋匠换粮食——交换是财富基础

```yaml
id: case-c01
title: 鞋匠用鞋子换粮食
type: case
source_chapter: 第1章「财富的本质」
source_quote: |
  你是村里有名的鞋匠，但是最近家里的粮食快吃完了……你是个鞋匠啊，你是不是可以用鞋子去换吃的呢？你做了双新鞋子，然后去村子里挨家挨户地推销，终于你找到了一家，他们正好需要鞋子，也正好有多余的食物，你们很快达成了交易。你看，这就是开头我说的："财富的基础是交换！"
summary: 5000年前农耕文明中，鞋匠通过做鞋子与他人交换粮食和衣服，发现只要不断做鞋就能换到任何想要的东西，由此揭示了财富的基础是交换这一核心原理。
bound_to: 财富基础=交换（第1章核心方法论）
outcome: 鞋匠通过持续交换让家庭变得越来越富裕，证明了个人生产总值=时间单价×能产生价值的时间这一公式的前提——价值必须通过交换才能变现。
tags: [交换, 财富本质, 个人生产总值, 比较优势]
```

## case-c06: 小和尚卖石头——价值由需求+稀缺性决定

```yaml
id: case-c06
title: 小和尚在不同场所卖同一块石头
type: case
source_chapter: 第8章「你的价值由什么决定」
source_quote: |
  小和尚从庙里搬了一块大石头到菜场，大婶出价20元买去压咸菜；师父让他去博物馆，艺术家出价2000元用来雕刻神像；再拿到古董店，土豪出价20万问这是哪个朝代的。同样一块石头，在不同地方有完全不同的价格。为什么？第一，得有人想买（需求）；第二，场景决定了稀缺性感知。
summary: 同一块普通石头在菜场只值20元、在博物馆值2000元、在古董店被出价20万。作者由此提炼出价值两大要素：有人需要（需求）和场景营造的稀缺感知。还补充说明如果多个小和尚都卖石头，价格会立刻下跌——稀缺性被打破。
bound_to: 价值=需求×稀缺性（第8章核心方法论）
outcome: 证明价格不由物品本身决定，而由"是否有需求"和"是否稀缺"共同决定，为个人提升价值提供了清晰方向。
tags: [价值定价, 需求, 稀缺性, 场景, 可替代性]
```

## case-c13: 马斯克火星殖民分解——乘法分解

```yaml
id: case-c13
title: 马斯克火星移民计划的乘法分解
type: case
source_chapter: 第22章「做计划」
source_quote: |
  马斯克有一个疯狂的火星移民计划：把100万人送上火星。目前把一个人送上火星的成本约100亿美元。马斯克的目标是把成本降到50万美元，即降低2万倍。然后他把目标拆分成一个神奇的公式：20000=20×10×100。20=载人能力提高20倍（一次运送100人）；10=发射成本降低10倍（自行生产零件）；100=火箭重复使用100次（已实现火箭回收）。一个看似完全不靠谱的目标，被分解成几个有乘法关系的小目标之后，就变得可解决了。
summary: 马斯克的火星移民计划看似天方夜谭——需将每人运输成本从100亿美元降至50万美元（降低2万倍）。他将2万倍拆解为20×10×100三个乘法因子，每个因子对应一个看似困难但可行的子目标，整体竟变得可解。
bound_to: 乘法分解（第22章核心方法论——用数学公式拆解指数级目标）
outcome: 通过乘法分解，一个"不可能"的目标被拆解为三个各自可实现的子目标，SpaceX已在火箭回收等环节取得突破性进展。
tags: [乘法分解, 目标拆解, 马斯克, SpaceX, 指数思维]
```
